Acabo de publicar un nuevo demo de RAG con OpenShift AI y LangChain4j.

El proyecto muestra cómo desplegar un pipeline completo de Retrieval Augmented Generation usando:
- OpenShift AI para el modelo
- Milvus como vector store
- Quarkus + LangChain4j para la API

Todo containerizado y listo para producción.
